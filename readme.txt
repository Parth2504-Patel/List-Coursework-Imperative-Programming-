This program converts denary into binary and vice versa.
It works for Unsigned: Char and Int
                      Signed: Char, Int and long
This will only be able to take one conversion at a time.
To run the tests, enter ./visualise
To convert into binary, enter the denary value ./visualise char 56
To convert into denary, please enter the binary in nibbles ./visualise char 0111 1001
